*Status: DEV*

Development Workspace
=====================

**NOTE: The tools used by this workspace are still under development.**

Requirements:

  * [node.js](http://nodejs.org/)
  * [sm](https://github.com/sourcemint/sm)


Commands
========

Run in dev mode:

    make run

Build for distribution:

    make build

Publish new release:

    make publish
